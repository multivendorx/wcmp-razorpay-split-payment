== Description ==

The much awaited <strong>WCMp Razorpay Checkout Payment gateway</strong> is now live.

<strong>WCMp Razorpay Checkout Gateway</strong> is a <strong>WC Marketplace</strong> add-on that allows instant (chain or parallel) payment to vendors.

<strong>Admin can</strong>

Activate or deactivate Razorpay Checkout payment for customers as well vendors.


To know more about WCMp Razorpay Checkout Gateway, please <a href="http://wc-marketplace.com/knowledgebase/wcmp-razorpay-Checkout-gateway/">visit us</a>

== Installation ==

NOTE:  WCMp Razorpay Checkout Gateway plugin is an extension of WooCommerce as well as WC Marketplace. As such, Woocommerce and WC Marketplace plugin 
needs to be installed and activated on your WordPress site for this plugin to work properly.

1. Download and install Woocommerce
2. Download and install WC Marketplace
4. Download and install WCMp Razorpay Checkout Gateway plugin using the built-in Word Press plugin installer. If you download WCMp Razorpay Checkout Gateway plugin 
   manually, make sure that it's uploaded to /WP-content/plug ins/ and activate the plug in from the Plugin menu from your WordPress dashboard.      
   Alternatively, follow these steps below and install the addon: 
   Plugins > Add new > Upload plugin > Upload wcmp-razorpay-checkout-gateway.zip > Install Now > Activate Plug in.
5. Configure the checkout settings from WooCommerce > Settings > Checkout tab > Razorpay Checkout sub tab.
6. Configure marketplace features from Woocommerce > WCMp > Payment tab > Razorpay Checkout sub tab.
